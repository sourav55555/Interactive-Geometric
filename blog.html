What is DOM and what is the purpose of DOM?

Which can convert our HTML document to an object-like model called DOM.

With DOM object module, javascript can create dynamic HTML. DOM maintain a standard programming interface for HTML document. With DOM javascript can create, navigate and modify any element in the document.

How will you select HTML elements using DOM? Name the DOM methods.

DOM selectors use JavaScript to select the HTML elements. There are 6 ways for selecting HTML elements with JavaScript.

1. getElementsByTagName() -> the method returns the elements that match the passed tag name.
2. getElementsByClassName() -> the method returns the elements that match the passed class name.
3. getElementById() -> the method returns an element whose id matches.
4. querySelector() -> the method returns the first element that matches the passed selector.
5.querySelectorAll() -> the method returns all the elements that match the passed selector.
6. getElementsByName() -> the method returns the elements that match the value of the name attribute with the passed string.

What is event bubble?

When an element receives an event, that event bubbles up to its parent, then parent to grandparents and so on until it gets the root of the element in the DOM. It's called an event bubble.

What is a callback function and why will you use it?

We can pass parameters in a function, and if the passed parameter is another function so that the function can be executed is called a callback function.

JavaScript runs codes from top to down order. If a function is defined before a function but I need to execute the function right after something has happened or some task is completed then I can use the callback function.
